# Aura Thoughts

A simple and elegant FiveM resource for displaying dynamic thought bubbles in-game.

## Documentation

For detailed installation, usage, and configuration instructions, please refer to our [official documentation](https://aura-development.gitbook.io/aura-development-docs/documentation/free-releases/aura_thoughts).

## Quick Start

1. Download and place in `resources/`
2. Add `ensure aura-thoughts` to `server.cfg`
3. Configure zones in `config.lua`
4. Restart server

## Features

- Zone-based contextual thoughts
- Customizable icons and colors
- Persistent and temporary display modes
- Modern UI with smooth animations
